# Brain-to-Text 25

This repository contains code and resources for the Kaggle **Brain-to-Text 25** competition. It includes data preprocessing scripts, model implementations, experiment notebooks, and documentation for collaborating on the project.

## Repository Structure

- `data/` – Contains raw and processed datasets (do not commit large data files).
- `notebooks/` – Jupyter notebooks for exploration and experiments.
- `src/` – Source code organized by module (data, models, features, etc.).
- `scripts/` – Shell scripts for training, evaluation, and inference.
- `outputs/` – Logs, checkpoints, and submission files.
- `docs/` – Documentation and experiment logs.
