# Data Directory

This directory contains raw and processed datasets for the Brain-to-Text 25 competition.

- `raw/` – Downloaded data from the Kaggle competition. Do **not** commit large dataset files to the repository.
- `processed/` – Preprocessed or feature-engineered versions of the dataset used for experiments.

Use the `setup.sh` script to download the competition data into the `raw/` directory and extract it.
